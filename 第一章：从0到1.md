# 第1章读书笔记 - 《从零到一》

## 1.1 学习计划
*   **目标**：教授如何设计和构建微处理器
*   **学习路径**：逻辑门 → 组合模块（如加法器） → 汇编语言 → 微处理器
*   **主要主题**：管理复杂性

## 1.2 管理复杂性的艺术

### 抽象
* 在不重要时隐藏细节
* **计算机系统的抽象层次**：
  - 物理层 → 器件层 → 模拟电路 → 数字电路 → 逻辑设计 → 微体系结构 → 体系结构 → 操作系统 → 应用软件

### 规范
* 限制设计选择以提高工作效率
* **数字规范**：使用离散电压（0/1）而非连续模拟电压

### 三个"-Y"原则
* **层次化**：将系统划分为模块和子模块
* **模块化**：模块具有明确定义的功能和接口
* **规整化**：重用模块化组件

## 1.3 数字抽象
* **离散值变量**：有限数量的值，与连续物理变量相对
* **二进制表示**：区分两个电压比区分十个电压更容易
* **信息度量**：`D = log₂N 比特`
* **二进制变量**：携带1比特信息
* **布尔逻辑**：在二进制变量上运算（真/假，1/0，高/低）

## 1.4 数字系统

### 十进制（基数10）
* 数字：0-9
* 列权重：10的幂

### 二进制（基数2）
* 数字：0, 1
* 列权重：2的幂（1, 2, 4, 8, 16...）

### 十六进制（基数16）
* 数字：0-9, A(10), B(11), C(12), D(13), E(14), F(15)
* 每个十六进制数字 = 4个二进制数字（半字节）

### 数据单位
* **比特**：二进制数字
* **半字节**：4比特
* **字节**：8比特
* **字**：取决于处理器（32或64比特）

### 二进制加法
* 类似于十进制加法
* 当列的和为2或3（10₂ 或 11₂）时传播进位位

### 有符号二进制数

#### 符号幅度表示法
* 最高有效位 = 符号（0=正，1=负）
* 剩余位 = 幅度
* 零有两种表示（+0 和 -0）

#### 二进制补码
* 最高有效位具有负权重
* **N位数范围**：`[–2^(N–1) 到 2^(N–1) – 1]`
* **符号反转**：所有位取反，然后加1
* **溢出**：同号相加产生异号结果时发生
* **符号扩展**：将符号位复制到所有新的最高有效位位置

## 1.5 逻辑门
* **非门**：`Y = A̅`
* **与门**：`Y = A • B`
* **或门**：`Y = A + B`
* **异或门**：`Y = A ⊕ B`（奇数个输入为真时输出为真）
* **与非门**：`Y = A • B‾`
* **或非门**：`Y = A + B‾`
* **同或门**：`Y = A ⊕ B‾`（相等门）

## 1.6 数字抽象之下

### 逻辑电平
* **输出高电平**：门输出高电平的最小电压
* **输出低电平**：门输出低电平的最大电压
* **输入高电平**：门输入解释为高电平的最小电压
* **输入低电平**：门输入解释为低电平的最大电压
* **禁区**：VIL和VIH之间的区域

### 噪声容限
* **低电平噪声容限** = `VIL – VOL`
* **高电平噪声容限** = `VOH – VIH`

### 静态规范
* 门必须在给定有效输入时产生有效输出
* 支持构建更大系统

## 1.7 CMOS晶体管

### nMOS晶体管
* 当栅极 = 1（高电平）时导通
* 能良好传递0（低电平）但弱传递1（高电平）

### pMOS晶体管
* 当栅极 = 0（低电平）时导通
* 能良好传递1（高电平）但弱传递0（低电平）

### CMOS
* 使用互补的nMOS和pMOS网络
* 静态功耗低

## 1.8 功耗

### 动态功耗
* 信号切换时对电容充电消耗的功率
* 公式：`P_动态 = α C V_DD² f`
  - `α` = 活动因子
  - `C` = 电容
  - `V_DD` = 电源电压
  - `f` = 频率

### 静态功耗
* 系统空闲时消耗的功率（漏电流）
* 公式：`P_静态 = I_DD V_DD`
  - `I_DD` = 静态电源电流
